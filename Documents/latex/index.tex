\hypertarget{index_overview}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{index_overview}
{\bfseries{ESP Weather Dashboard}} is a modern IoT monitoring system built using\+:


\begin{DoxyItemize}
\item {\bfseries{Streamlit (Frontend)}} — for real-\/time data visualization
\item {\bfseries{Fast\+API (Backend)}} — for data collection and REST API services
\item {\bfseries{ESP32 / ESP8266 Nodes}} — for temperature and humidity sensing
\end{DoxyItemize}

Each ESP node periodically sends JSON data to the Fast\+API server. ~\newline
 The Streamlit dashboard then displays these values live in an elegant {\bfseries{dark, weather-\/themed interface}}.
