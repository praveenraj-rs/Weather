Detailed Description

Streamlit frontend dashboard for ESP Weather Monitoring System.

Streamlit-based IoT dashboard for ESP sensor data and OpenWeather API.

This Streamlit application continuously fetches weather data from the backend REST API and displays live temperature and humidity readings from multiple ESP nodes.

It uses a custom dark UI theme, dynamic weather icons, and auto-refresh functionality to provide a user-friendly IoT dashboard.

This Streamlit frontend visualizes live environmental data collected from ESP-based IoT nodes and integrates real-time weather information using the OpenWeather API. The dashboard displays:

  • Temperature, humidity, and status from multiple ESP devices
  • Live weather data for a selected city
  • Local date and time display
  • A GitHub footer link for project reference

The application automatically refreshes every 5 seconds to ensure real-time updates.
